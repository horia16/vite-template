# Project modules

This template takes on a modular approach to creating an application.
This allows for better readability and code management.

## Module structure

Each module should follow this structure:

```bash
|- modules
    |- my-example-module
        |- assets
        |- components
        |- locales
        |- routes
        |- store
        |- index.vue
    |- my-other-module
        |- ...
```
